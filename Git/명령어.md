# .git 디렉토리

git의 핵심
- 간단한 키 - 값 데이터 저장소이다.

.git/objects 디렉토리
- 객체 데이터베이스 저장소
    - objects/pack
    - objects/info
    - git hash-object
        - hash-object 내용을 고유 키 값으로 반환
                - SHA-1 해시 사용
    - 오브젝트 유형 : blob
- 트리 객체

.git/refs
- SHA-1 값을 읽기 쉬운 이름으로 저장
- /heads : branch
- /tags : tag
- refs/remotes

.git/HEAD
- 현재 위치한 포인터에 대한 상징적인 참조를 나타낸다.
